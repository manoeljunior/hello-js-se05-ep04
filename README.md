# Exercicio 04

Criar as operações INSERT, DELETE, UPDATE e SELECT, utilizando **knex**, **node** e **sqlite**.

## Como executar?

#### INSERT
```bash
> node index.js INSERT Sofia,32232323
```

#### UPDATE
```bash
> node index.js UPDATE 1,Sofia,92323232
```

#### DELETE
```bash
> node index.js DELETE 2
```

#### SELECT
```bash
> node index.js SELECT
```
